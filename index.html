Project Overview

SmartIrrigate Pro is an advanced IoT-based soil irrigation system that combines real-time sensor monitoring with machine learning predictions to optimize water usage. Built on a mobile 4-wheel chassis platform, this autonomous system can navigate and monitor multiple areas while providing precise watering recommendations.

ESP32 Powerhouse

Dual-core processor with WiFi/Bluetooth for sensor management and data transmission

DHT11 Sensor

Accurate temperature and humidity monitoring for environmental analysis

Soil Moisture Sensor

Capacitive sensor for precise soil moisture detection without corrosion

ML Prediction Engine

AI-powered algorithm calculates optimal water dispersion based on sensor data

Mobile Chassis

4-wheel platform controlled by NodeMCU with FS-CT6B remote for navigation

Real-Time IoT

WebSocket communication for instant data updates and remote control

How to Use

Step-by-step guide to get started

01

Hardware Setup

  • Mount ESP32 and sensors on the 4-wheel chassis
  • Connect DHT11 sensor to ESP32 (Data pin to GPIO 4)
  • Connect soil moisture sensor to ESP32 (Analog pin to GPIO 34)
  • Connect NodeMCU to L298N motor driver for chassis control
  • Pair FS-CT6B remote (2.4GHz) with receiver for RC navigation
  • Ensure proper power supply (7.4V LiPo battery recommended)
02

Software Configuration

  • Download Arduino IDE and install ESP32 board support
  • Install required libraries: DHT, WiFi, WebServer, ArduinoJson
  • Open the ESP32 firmware code and update WiFi credentials
  • Set your local server IP address in the configuration
  • Upload the code to ESP32 via USB
  • Upload NodeMCU chassis control code separately
  • Verify serial monitor shows successful WiFi connection
03

Network Connection

  • Ensure ESP32 and your computer are on the same WiFi network
  • Note the IP address shown in the serial monitor
  • Open the web interface by entering the IP in your browser
  • Update the ESP32_IP in config.js file if needed
  • Check the connection status indicator turns green
  • Verify real-time data starts flowing to the dashboard
04

Monitor & Control

  • View real-time sensor readings on the dashboard
  • Monitor soil moisture, temperature, and humidity levels
  • Check ML model water recommendations (display only)
  • Use displayed water amounts to manually irrigate as needed
  • Use FS-CT6B remote for full chassis navigation control
  • Review historical data charts for trend analysis
  • Set up alerts for critical moisture levels (optional)
05

Maintenance & Calibration

  • Calibrate soil moisture sensor in dry and wet soil
  • Clean sensor probes weekly to ensure accuracy
  • Check battery voltage regularly (min 6.5V)
  • Update ML model parameters based on crop type
  • Test FS-CT6B remote control range and responsiveness
  • Verify chassis motors and wheels for smooth operation
  • Keep firmware updated for latest features

Troubleshooting Tips

No WiFi Connection:

Check credentials, router settings, and ESP32 antenna placement

Inaccurate Readings:

Recalibrate sensors, check wiring connections, and verify power supply

Chassis Not Responding:

Check motor driver connections, battery level, and remote pairing

Display Mode Active:

System shows water recommendations only. No automatic irrigation - apply water manually based on displayed amounts.
