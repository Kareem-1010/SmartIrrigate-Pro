## ğŸ› Troubleshooting

### Connection Issues

**Problem**: Can't connect to ESP32
```
Solution:
1. Check WiFi credentials
2. Verify ESP32 is powered on
3. Check serial monitor for IP address
4. Ensure device is on same network
5. Ping ESP32 IP to test connectivity
```

**Problem**: WebSocket disconnects frequently
```
Solution:
1. Check WiFi signal strength
2. Reduce distance to router
3. Lower update frequency in config
4. Check power supply stability
```

### Sensor Issues

**Problem**: Inaccurate moisture readings
```
Solution:
1. Calibrate sensor (dry vs wet soil)
2. Clean sensor contacts
3. Check wiring connections
4. Verify voltage (3.3V or 5V)
5. Replace sensor if corroded
```

**Problem**: DHT11 returns NaN
```
Solution:
1. Add 10kÎ© pull-up resistor to DATA pin
2. Use longer delay between readings (2 seconds)
3. Check power supply
4. Try different GPIO pin
5. Replace sensor if faulty
```

### Chassis Issues

**Problem**: Motors not responding
```
Solution:
1. Check battery voltage (min 6.5V)
2. Verify motor driver connections
3. Test motors directly with battery
4. Check FS-CT6B pairing
5. Upload fresh NodeMCU code
```

**Problem**: Remote control not working
```
Solution:
1. Replace transmitter batteries
2. Re-pair receiver with transmitter
3. Check receiver connections
4. Verify PWM signal on NodeMCU
5. Test with different channel
```

### Pump Issues

**Problem**: Water pump not activating
```
Solution:
1. Test relay manually (apply 5V to IN pin)
2. Check GPIO pin configuration
3. Verify pump voltage requirements
4. Test pump with direct power
5. Check relay module orientation
```
